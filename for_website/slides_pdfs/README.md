# Slide PDFs

Module slides are built with **reveal.js** (HTML + Markdown). To get PDFs for offline sharing or printing:

1. **From the repo** (or from `slides_source/` in this folder if you copied it):
   - Open each module’s `index.html` in a **browser** (Chrome or Edge work well).
   - Use **Print** (Ctrl/Cmd + P) → **Destination: Save as PDF**.
   - Save as e.g. `module-01-intro-genai.pdf`, `module-02-core-prompt-engineering.pdf`, etc.

2. **Place the PDFs here** so your website can link to them (e.g. `module-01-intro-genai.pdf`, …).
